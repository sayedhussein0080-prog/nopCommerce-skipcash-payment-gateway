### Why choose this plugin over the free version?
- **Flawless Asynchronous Webhooks:** Includes fully tested SkipCash Webhook integration to ensure order statuses update behind the scenes even if the user drops off the payment loading screen.
- **Premium User-Interface CSS:** Completely overrides the default NopCommerce checkout radio-button list with a highly interactive, responsive Flexbox Card GUI displaying beautiful SkipCash logos.
